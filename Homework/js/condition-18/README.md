# js-lab-35
### Lab35 Conditiona: Refractor4
เปลี่ยน if-else ข้างล่างให้อยู่ในรูปของ Ternary Operators

```JavaScript
let login = prompt('Enter username');
let message;
if (login == 'Employee') {
  message = 'Hello';
} else if (login == 'Director') {
  message = 'Greetings';
} else if (login == '') {
  message = 'No login';
} else {
  message = '';
}
```
- ศิรวิชญ์ ประสิทธิเวชานนท์ (เอม)
