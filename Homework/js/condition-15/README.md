### Lab32 Conditiona: Refractor2
จงเขียนโค้ดในรูปแบบ if else

```JavaScript
switch (browser) {
  case 'Edge':
    alert(""You've got the Edge!"");
    break;
  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert('Okay we support these browsers too');
    break;
  default:
    alert('We hope that this page looks ok!');
}
```
